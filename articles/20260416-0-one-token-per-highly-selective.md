## Resumo

O artigo "One Token per Highly Selective Frame: Towards Extreme Compression for Long Video Understanding" aborda o desafio da compreensão de vídeos longos por modelos de visão-língua (VLMs). Devido ao grande número de frames em vídeos, os VLMs precisam perceber os frames de forma esparso, perdendo informações temporais. Para superar isso, o artigo propõe uma compressão de tokens extrema, onde cada frame é representado por um único token na camada final do modelo de linguagem (LLM).

A compressão de tokens é alcançada por meio de módulos progressivos e aprendíveis, que permitem que o VLM digira 2-4 vezes mais frames com desempenho melhorado. Além disso, a compressão de frames é investigada, selecionando os frames mais relevantes para as perguntas com base nos escores de atenção interna do LLM. Essa abordagem, chamada de compressão condicionada à pergunta (QC-Comp), permite uma compressão ainda maior.

Para mitigar a bias de posição da atenção do LLM em contextos longos, os vídeos são divididos em segmentos curtos e empregada atenção local. O modelo proposto, chamado de \name, alcança uma compressão extrema e permite uma amostragem mais densa de frames. O modelo é treinado de forma eficiente, com uma etapa de ajuste de compressão supervisionada que utiliza apenas 2,5% dos dados de treinamento supervisionado, aumentando a precisão em 3,3% em relação ao modelo de base.
